# Windows Basics

## Introduction to Microsoft Windows

### Overview

Microsoft Windows is one of the world's most widely used operating systems, powering millions of personal computers, business workstations, and enterprise environments. It provides a user-friendly graphical interface that allows users to interact with files, applications, hardware, and system settings efficiently.

For cybersecurity professionals, understanding Windows is essential because it is a common target for security assessments, system administration tasks, incident response activities, and enterprise security operations.

In this room, you take on the role of a new employee at TryHatMe, learning how to navigate and manage a Windows workstation while becoming familiar with the tools and features used in daily operations.

---

## Main Concept

### What is Microsoft Windows?

Microsoft Windows is a graphical operating system developed by Microsoft that provides a platform for running applications, managing hardware resources, organizing files, and securing user data.

Windows allows users to interact with the computer through a Graphical User Interface (GUI), making it accessible to both technical and non-technical users.

Common Windows Versions:

| Version             | Status                |
| ------------------- | --------------------- |
| Windows 10          | Legacy / End of Life  |
| Windows 11          | Current Desktop OS    |
| Windows Server 2019 | Enterprise Server     |
| Windows Server 2022 | Enterprise Server     |
| Windows Server 2025 | Latest Server Release |

## Important Components

## Windows Desktop

### What is the Desktop?

The Desktop is the primary workspace displayed after a user logs into Windows.

It provides access to:

* Applications
* Files
* Folders
* System tools

### Common Desktop Elements

| Component         | Purpose                                |
| ----------------- | -------------------------------------- |
| Icons             | Quick access to files and applications |
| Taskbar           | Displays running programs              |
| Start Menu        | Central navigation hub                 |
| Notification Area | Displays system status                 |

---

## Taskbar

The Taskbar is located at the bottom of the screen by default.

### Functions

* Access system settings

## File Explorer

### What is File Explorer?

File Explorer is Windows' built-in file management utility.

It allows users to:

* Browse files
* Create folders
* Move data
* Organize directories
* Access storage devices

### Common Locations

```text
This PC
Documents
Downloads
Pictures
Desktop
Network
```

### Example File Path

```text
C:\Users\Employee\Documents\Reports
```

---

## Understanding File Paths

A file path identifies the location of a file or folder.

### Example

```text
C:\Users\John\Desktop\Project\File.txt
```

### Path Components

| Component | Description      |
| --------- | ---------------- |
| C:        | Drive Letter     |
| Users     | Parent Directory |
| John      | User Profile     |
| Desktop   | Folder           |
| File.txt  | File Name        |

## Task Manager

### What is Task Manager?

Task Manager is a Windows administrative tool used to monitor system performance and running processes.

### Functions

* View active applications
* Monitor CPU usage
* Monitor RAM usage
* Manage startup applications
* End unresponsive processes

### Common Tabs

| Tab         | Purpose                      |
| ----------- | ---------------------------- |
| Processes   | Running applications         |
| Performance | Resource usage               |
| Startup     | Startup programs             |
| Users       | Logged-in users              |
| Details     | Advanced process information |

### Opening Task Manager

```text
Ctrl + Shift + Esc
```

or

```text
Right Click Taskbar
→ Task Manager
```

## Windows Security

### What is Windows Security?

Windows Security is Microsoft's built-in security platform that protects the operating system from threats.

### Core Security Features

#### Virus & Threat Protection

Provides:

* Malware scanning
* Real-time protection
* Threat detection

---

#### Firewall & Network Protection

Monitors:

* Incoming connections
* Outgoing connections
* Network traffic

---

#### Account Protection

Secures:

* User accounts
* Sign-in methods
* Authentication settings

---

#### Device Security

Protects:

* Hardware integrity
* Secure boot functionality
* Trusted Platform Module (TPM)

---
