# Linux Fundamentals Part 1 — Project Report

## Objective

The objective of this task was to develop foundational Linux filesystem navigation skills by using commands to list directories, navigate the filesystem, read file contents, and identify the current working directory.

---

## Tasks Completed

### Challenge 1

**Question:**

On the Linux machine that you deploy, how many folders are there?

**Answer:**

```text
4
```

---

### Challenge 2

**Question:**

Which directory contains a file?

**Answer:**

```text
folder4
```

---

### Challenge 3

**Question:**

What is the contents of this file?

**Answer:**

```text
Hello World
```

---

### Challenge 4

**Question:**

Use the cd command to navigate to this file and find out the new current working directory. What is the path?

**Answer:**

```text
/home/ubuntu/folder4
```

## Key Concepts Learned

* Using `ls` to list files and folders.
* Using `cd` to navigate directories.
* Using `cat` to display file contents.
* Using `pwd` to determine the current directory.
* Understanding absolute filesystem paths.
* Navigating Linux systems without a graphical interface.
