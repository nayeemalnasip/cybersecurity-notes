# Linux Fundamentals Part 1 — Project Report

## Objective

The objective of this task was to introduce basic Linux terminal interaction and familiarize users with essential commands used to communicate with the operating system.

---

## Tasks Completed

### Challenge 1

**Question:**

If we wanted to output the text "**TryHackMe**", what would our command be?

**Answer:**

```bash
echo TryHackMe
```

---

### Challenge 2

**Question:**

What is the username of who you're logged in as on your deployed Linux machine?

**Answer:**

```text
tryhackme
```

## Key Concepts Learned

* Linux systems are commonly managed through the terminal.
* The `echo` command outputs text to the console.
* The `whoami` command identifies the currently logged-in user.
* Understanding the terminal is essential for cybersecurity operations.
* Basic commands form the foundation for advanced Linux administration and penetration testing.

## Conclusion

This task provided hands-on experience with the Linux terminal and introduced two fundamental commands: `echo` and `whoami`.
