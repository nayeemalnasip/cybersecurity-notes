# Linux Fundamentals Part 1 — Project Report

## Objective

The objective of this task was to learn how to efficiently locate files and search file contents within Linux systems using the `find` and `grep` commands. These skills are fundamental for system administration, security investigations, penetration testing, and log analysis.

---

## Tasks Completed

### Challenge 1

**Question:**

Use grep on `access.log` to find the flag that has a prefix of `THM`. What is the flag?

**Answer:**

```text
THM{ACCESS}
```
