## Key Concepts Learned

* DNS translates domain names into IP addresses.
* HTTP and HTTPS facilitate communication between clients and web servers.
* Load Balancers distribute traffic and perform health checks.
* CDNs improve performance by serving content from geographically closer servers.
* Databases store and retrieve application data.
* WAFs protect web applications from common attacks.
* Virtual Hosts allow multiple websites to run on a single web server.
* Static content remains unchanged across requests.
* Dynamic content is generated by backend applications.
* Backend code is processed server-side and is not visible to clients.
* Modern web applications rely on multiple interconnected technologies.
