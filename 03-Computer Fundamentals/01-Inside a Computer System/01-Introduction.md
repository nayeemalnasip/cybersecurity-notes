# Inside a Computer System

## Introduction to Computer Systems

### Overview

Before learning cybersecurity, it is essential to understand the system we are trying to protect. A computer system is made up of multiple hardware and software components that work together to process data, store information, and provide services to users. Understanding these components and their interactions forms the foundation for studying system security, networking, operating systems, and ethical hacking.

Just as a castle defender must understand the castle's layout before protecting it, a cybersecurity professional must understand a computer's internal structure before securing it.

---

## Main Concept

### What is a Computer System?

A Computer System is an electronic device that receives input, processes data, stores information, and produces output.

It consists of several interconnected components that work together to perform tasks requested by users.

Basic Computer System Flow:

```text
Input
  ↓
Processing
  ↓
Storage
  ↓
Output
```

Examples of computer systems include:

* Desktop Computers
* Laptops
* Servers
* Smartphones
* Tablets
* Embedded Systems

---

## How It Works

### Step 1: Input

The user provides data or commands to the computer.

Examples:

```text
Keyboard
Mouse
Touchscreen
Microphone
Scanner
```

---

### Step 2: Processing

The processor interprets and executes instructions.

Example:

```text
User opens a web browser
        ↓
CPU processes instructions
        ↓
Browser starts
```

---

### Step 3: Storage

Data is stored temporarily or permanently.

Examples:

```text
RAM (Temporary)
SSD/HDD (Permanent)
```

---

### Step 4: Output

The processed information is presented to the user.

Examples:

```text
Monitor
Printer
Speaker
Projector
```

---

## Important Components

### Hardware

Hardware refers to the physical parts of a computer system.

Examples:

```text
CPU
RAM
Motherboard
Storage Devices
Power Supply
Network Card
```

Characteristics:

* Physical components
* Can be touched
* Perform computing operations

---

### Software

Software consists of programs and instructions that tell hardware what to do.

Examples:

```text
Operating Systems
Applications
Drivers
Utilities
```

Characteristics:

* Intangible
* Controls hardware operations
* Provides user functionality

---

### Users

Users interact with computer systems to perform tasks.

Examples:

```text
Home Users
Students
Administrators
Developers
Security Analysts
```

Users provide input and receive output from the system.

---

### Data

Data is the information processed and stored by computers.

Examples:

```text
Documents
Images
Videos
Emails
Databases
Logs
```

Data is one of the most valuable assets that cybersecurity aims to protect.

---

### Communication Between Components

All computer components communicate with each other to complete tasks.

Example Workflow:

```text
Keyboard
   ↓
CPU
   ↓
RAM
   ↓
Storage
   ↓
Monitor
```

Each component has a specific role in the computing process.

## Advantages / Benefits

* Automates tasks efficiently
* Stores large amounts of information
* Processes data quickly
* Supports communication and collaboration
* Enables internet access

## Key Characteristics

* A computer system consists of hardware and software.
* Components work together to process information.
* Data flows through input, processing, storage, and output stages.
* Users interact with systems through input devices.
* Computers can store both temporary and permanent data.
* Every system component has a specific function.
* Understanding computer architecture is essential for cybersecurity.

## Example

### Opening a Web Browser

When a user opens a browser:

```text
User Clicks Browser Icon
          ↓
Operating System Receives Request
          ↓
CPU Processes Instructions
          ↓
RAM Loads Required Data
          ↓
Browser Launches
          ↓
Display Appears on Monitor
```

This simple action involves multiple components working together simultaneously.
