# What is the OSI Model?

## Overview

This task introduced the Open Systems Interconnection (OSI) Model, a fundamental networking framework that standardizes how devices communicate across networks. The OSI Model defines a structured approach for sending, receiving, and interpreting data, ensuring compatibility between different networking devices and technologies.

## Methodology / Steps

### 1. Understanding the OSI Model

The Open Systems Interconnection (OSI) Model is a networking framework that defines how data is transmitted between devices on a network.

Its primary purpose is to provide a standard method of communication that allows different devices and technologies to work together.

---

### 2. Importance of the OSI Model

The OSI Model ensures that devices with different hardware, software, and functions can communicate effectively.

Benefits include:

* Standardized communication processes.
* Compatibility between different network devices.
* Easier troubleshooting and network management.
* Improved understanding of network operations.

---

### 3. The Seven Layers of the OSI Model

The OSI Model consists of seven layers arranged from Layer 7 to Layer 1.

| Layer   | Name         |
| ------- | ------------ |
| Layer 7 | Application  |
| Layer 6 | Presentation |
| Layer 5 | Session      |
| Layer 4 | Transport    |
| Layer 3 | Network      |
| Layer 2 | Data Link    |
| Layer 1 | Physical     |

Each layer has specific responsibilities and contributes to the overall communication process.

---

### 4. Encapsulation

As data travels through the OSI layers, additional information is added at each stage.

This process is known as:

```text
Encapsulation
```

Encapsulation allows data to carry the information required for successful delivery and interpretation across a network.

---

### 5. Layer-Based Communication

When data is sent across a network:

1. It moves through the OSI layers.
2. Each layer performs specific tasks.
3. Additional information is added to the data.
4. The receiving device processes the data through the same layers in reverse order.

This standardized process ensures reliable communication between devices.
