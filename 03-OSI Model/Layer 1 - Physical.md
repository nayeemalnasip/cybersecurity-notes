# Layer 1 - Physical

## Overview

This task introduced the Physical Layer, which is the first and lowest layer of the OSI Model. The Physical Layer is responsible for the physical transmission of data between devices using networking hardware and electrical signals.

---

## Objectives

* Understand the purpose of the Physical Layer.
* Learn how data is physically transmitted across a network.
* Identify the hardware components associated with this layer.
* Understand the use of binary signals in network communication.

---

* No tools were used in this task.

## Methodology / Steps

### 1. Understanding the Physical Layer

The Physical Layer is Layer 1 of the OSI Model and represents the actual hardware involved in network communication.

It is responsible for transmitting raw data between devices through physical connections.

---

### 2. Data Transmission

At this layer, devices communicate by sending electrical signals across a physical medium.

Data is transmitted using a binary numbering system consisting of:

```text id="u8yq2m"
0s and 1s
```

These binary values are converted into electrical signals that travel between connected devices.

---

### 3. Physical Components

The Physical Layer includes the hardware used to establish network connectivity.

Examples include:

* Ethernet Cables
* Network Connectors
* Physical Ports
* Network Interface Hardware

These components provide the medium through which data travels.

---

### 4. Role of the Physical Layer

The primary responsibilities of the Physical Layer include:

* Physical transmission of data.
* Converting binary data into signals.
* Sending and receiving electrical signals.
* Establishing physical connectivity between devices.

---

### 5. Communication Process

The communication process at the Physical Layer follows these basic steps:

1. Data is converted into binary form.
2. Binary data is transformed into electrical signals.
3. Signals travel through physical media such as Ethernet cables.
4. The receiving device interprets the incoming signals as binary data.

## Key Findings / Notes

* The Physical Layer is Layer 1 of the OSI Model.
* It is the lowest layer in the OSI framework.
* Responsible for physical data transmission.

* Data is transmitted as binary values (0s and 1s).
* Ethernet cables are a common component of this layer.
