# Linux Fundamentals Part 1

## Main Concepts

### Where is Linux Used?

#### What is it?

Linux is one of the most widely deployed operating systems in the world. Although many users interact daily with Windows or macOS, Linux operates behind the scenes in numerous critical technologies and services.

Linux is popular because it is:

* Lightweight
* Stable
* Secure
* Open Source
* Highly Customizable

#### How It Works

Organizations deploy Linux-based systems because they require:

1. Reliable performance
2. Low hardware requirements
3. Strong security controls
4. Scalability for enterprise environments
5. Cost-effective infrastructure

#### Important Components

| Environment                | Linux Usage                                 |
| -------------------------- | ------------------------------------------- |
| Web Hosting                | Websites and web applications               |
| Enterprise Servers         | Business infrastructure and services        |
| Mobile Devices             | Android operating system                    |
| Automotive Systems         | Infotainment and vehicle control systems    |
| Retail Systems             | Point of Sale (PoS) terminals               |
| Industrial Control Systems | Sensors, automation, and monitoring         |
| Cloud Computing            | Virtual machines and containers             |
| Cyber Security             | Penetration testing and security monitoring |

### Linux Distributions (Distros)

#### What is it?

Linux is not a single operating system. Instead, it is a family of operating systems built around the Linux Kernel.

Different versions are known as Linux Distributions (Distros).

#### How It Works

A Linux distribution combines:

```text
Linux Kernel
+
System Utilities
+
Package Manager
+
Desktop Environment (Optional)
+
Applications
=
Linux Distribution
```

Each distribution is designed for specific purposes such as:

* Desktop Computing
* Server Administration
* Cyber Security
* Cloud Infrastructure
* Embedded Systems

#### Common Linux Distributions

| Distribution                    | Primary Use                               |
| ------------------------------- | ----------------------------------------- |
| Ubuntu                          | Desktop and Server Environments           |
| Debian                          | Stable Server Deployments                 |
| Kali Linux                      | Penetration Testing and Security Research |
| Red Hat Enterprise Linux (RHEL) | Enterprise Infrastructure                 |
| CentOS Stream                   | Enterprise Development                    |
| Arch Linux                      | Advanced Customization                    |
| Parrot OS                       | Security and Privacy Focused              |

#### Ubuntu in Cyber Security

For Linux Fundamentals, Ubuntu is used because:

* Beginner-friendly
* Large community support
* Extensive documentation
* Widely used in enterprise environments
* Suitable for both servers and desktops

A notable feature is that Ubuntu Server can operate with as little as:

```text
512 MB RAM
```

making it suitable for lightweight deployments.

## Real-World Example

```text
Ubuntu Server
Nginx / Apache
MySQL / PostgreSQL
PHP / Python
```
