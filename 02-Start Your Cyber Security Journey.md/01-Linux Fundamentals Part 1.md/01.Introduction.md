## Main Concepts

### Introduction to Linux

#### What is it?

Linux is an open-source operating system that manages hardware resources and provides services for software applications. It serves as an alternative to operating systems such as Windows and macOS.

Linux is known for:

* Stability
* Security
* Performance
* Flexibility
* Open-source development

#### How It Works

1. The Linux Kernel communicates directly with hardware.
2. The Shell provides a command-line interface for user interaction.
3. Applications run on top of the operating system.
4. Users execute commands to manage files, processes, and system resources.

#### Important Components

| Component   | Description                                               |
| ----------- | --------------------------------------------------------- |
| Kernel      | Core component that manages hardware and system resources |
| Shell       | Command-line interface used to interact with Linux        |
| File System | Organizes and stores files and directories                |
| User Space  | Environment where applications and user processes run     |
| Terminal    | Interface used to execute Linux commands                  |

### Linux Command Line Interface (CLI)

#### What is it?

The Command Line Interface (CLI) allows users to interact with Linux through text-based commands instead of graphical interfaces.

#### How It Works

1. Open a terminal session.
2. Enter commands.
3. Linux processes the command.
4. Output is displayed in the terminal.

#### Important Components

```bash
pwd     # Display current directory
ls      # List files and directories
cd      # Change directory
mkdir   # Create directory
touch   # Create file
```

---

### File System Navigation

#### What is it?

Linux uses a hierarchical file system structure where files and directories are organized in a tree-like format.

#### How It Works

1. Start from the root directory (`/`).
2. Navigate through directories using commands.
3. Access files and folders based on permissions.

#### Important Components

```text
/
├── home
├── etc
├── var
├── usr
├── root
└── tmp
```

Common Commands:

```bash
pwd
ls
cd
tree
```

### Shell Operators

#### What is it?

Shell operators allow users to combine commands, redirect output, and automate tasks efficiently.

#### Important Components

| Operator | Function                                  |                                |
| -------- | ----------------------------------------- | ------------------------------ |
| `>`      | Redirect output to file                   |                                |
| `>>`     | Append output to file                     |                                |
| `        | `                                         | Pipe output to another command |
| `&&`     | Execute next command if previous succeeds |                                |
| `;`      | Execute multiple commands sequentially    |                                |

Examples:

```bash
ls > files.txt
```

```bash
cat file.txt | grep admin
```

```bash
mkdir Test && cd Test
```
